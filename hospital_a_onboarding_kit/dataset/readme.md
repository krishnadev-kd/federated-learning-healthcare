This dataset contains 12,064 T1-weighted contrast-enhanced brain MRI images, categorized into four classes:
(1) Glioma Tumor
(2) Meningioma Tumor
(3) Pituitary Tumor
(4) No Tumor (Normal)
The dataset is organized into 80% training data and 20% testing data, making it suitable for machine learning and deep learning research in tumor detection and classification.

📊 Dataset Composition:
--------------------------------
Training Data (80%)
Glioma: 3,018 images
Pituitary: 2,504 images
Meningioma: 2,183 images
No Tumor: 1,945 images

Testing Data (20%)
Glioma: 755 images
Pituitary: 546 images
Meningioma: 626 images
No Tumor: 487 images
